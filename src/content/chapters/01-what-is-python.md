## 파이썬은 어떤 언어인가요?

파이썬(Python)은 읽기 쉬운 코드를 지향하는 프로그래밍 언어입니다. 다른 언어들에 비해 문법이 간결해서 "실행 가능한 의사코드(executable pseudocode)"라고 불리기도 합니다.

우리 프로젝트(ASS-to-LRC 변환기)는 파이썬으로 만들어져 있습니다. 이 튜토리얼을 끝까지 따라가면, 프로젝트의 모든 코드를 읽고 이해할 수 있게 됩니다.

## .py 파일이란?

파이썬 코드는 `.py` 확장자를 가진 텍스트 파일에 작성합니다. 예를 들어 `hello.py`라는 파일을 만들면, 터미널에서 `python` 명령어로 실행할 수 있습니다.

> 비유하면: `.py` 파일은 **요리 레시피**와 같습니다. 레시피(코드)를 작성하고, 요리사(파이썬 인터프리터)에게 건네주면, 요리사가 한 줄씩 읽으며 요리(실행)를 합니다.

## Hello World 만들기

모든 프로그래밍 학습은 "Hello World"에서 시작합니다. 파일 하나를 만들어봅시다.

### 1단계: 파일 만들기

아무 텍스트 편집기로 `hello.py` 파일을 만들고, 다음 한 줄을 입력합니다:

```python
print("Hello, World!")
```

### 2단계: 실행하기

터미널을 열고 다음 명령어를 실행합니다:

```
$ python hello.py
Hello, World!
```

`print()`는 화면에 텍스트를 출력하는 **내장 함수(built-in function)**입니다. 괄호 안에 출력하고 싶은 내용을 넣으면 됩니다.

## 파이썬의 실행 방식: 인터프리터

파이썬은 **인터프리터 언어**입니다. 코드를 위에서 아래로, 한 줄씩 읽으며 실행합니다.

```python
print("첫 번째 줄")
print("두 번째 줄")
print("세 번째 줄")
```

실행하면:

```
첫 번째 줄
두 번째 줄
세 번째 줄
```

위에서 아래로 순서대로 출력됩니다. 이것이 파이썬의 가장 기본적인 동작 방식입니다.

## 우리 프로젝트에서의 실행

우리 프로젝트의 ASS-to-LRC 변환기도 같은 방식으로 실행합니다:

```
$ python ass2lrc.py input/example.ass output/example.lrc
```

`ass2lrc.py`는 `.py` 파일이고, 뒤에 오는 `input/example.ass`와 `output/example.lrc`는 **명령줄 인자(command-line arguments)**입니다. 프로그램에 "이 파일을 변환해서 여기에 저장해줘"라고 알려주는 것이죠.

## 주석(Comment)

코드에 설명을 달고 싶을 때는 `#` 기호를 사용합니다. `#` 뒤에 오는 내용은 파이썬이 무시합니다.

```python
# 이 줄은 주석입니다. 실행되지 않습니다.
print("이 줄은 실행됩니다")  # 줄 끝에도 주석을 달 수 있습니다
```

우리 프로젝트에서는 한국어 주석을 사용하고 있습니다:

```python
# ASS 태그를 모두 제거한 순수 텍스트 반환
clean_text = TAG_PATTERN.sub('', text)
```

## `if __name__ == '__main__':` 이건 뭔가요?

우리 프로젝트의 파일들 맨 아래에 이런 코드가 있습니다:

```python
if __name__ == '__main__':
    # 여기에 실행 코드
    main()
```

이것은 **"이 파일이 직접 실행될 때만 이 코드를 실행해라"**라는 의미입니다.

- `python ass2lrc.py`처럼 직접 실행하면 → `__name__`이 `'__main__'`이 되어 실행됨
- 다른 파일에서 `import ass2lrc`로 불러오면 → `__name__`이 `'ass2lrc'`가 되어 실행 안 됨

> 비유하면: 요리 레시피 책에서 "이 레시피를 직접 만들 때만 아래 재료를 준비하세요"라는 메모 같은 것입니다. 다른 레시피의 일부로 참고만 할 때는 무시하라는 뜻이죠.

## 핵심 정리

| 개념 | 설명 |
|------|------|
| `.py` 파일 | 파이썬 코드가 담긴 텍스트 파일 |
| `print()` | 화면에 출력하는 내장 함수 |
| `#` | 주석 — 실행되지 않는 설명 |
| `python 파일명.py` | 터미널에서 파이썬 파일 실행 |
| `__name__ == '__main__'` | 직접 실행할 때만 동작하는 가드 |

3. 주석을 달아보세요 — `#` 뒤의 내용이 정말 실행되지 않는지 확인!
